Proxy CORS
Navegadores bloqueiam chamadas diretas à API do Superlógica por CORS. Configure um proxy simples no Railway ou Render (veja instruções abaixo) e cole a URL aqui.
COMO CRIAR O PROXY (1x, 5 minutos)
1. Acesse railway.app → conta gratuita
2. New Project → Deploy from GitHub → use o template superlogica-proxy
3. Ou rode localmente: npx superlogica-proxy
4. Cole a URL gerada no campo acima
